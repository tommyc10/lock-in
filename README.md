# Lock In

A refined productivity app for building better habits and tracking daily progress.

---

## Features

- **Habit Tracking** — Daily habits organized by morning, afternoon, and evening
- **Workout Logger** — Log exercises, sets, reps, and weights
- **Vision Board** — Visualize your goals and aspirations
- **Countdown Events** — Track important upcoming dates
- **Daily Reflections** — Record wins, struggles, and plans

---

## Quick Start

```bash
git clone https://github.com/tommyc10/lock-in.git
cd lock-in
npm install
npm run dev
```

---

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- localStorage
